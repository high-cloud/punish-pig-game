# 制裁臭猪大作战 🐷

一个有趣的点击惩罚小游戏！

## 游戏特色

- 30秒限时挑战
- 随机出现的臭猪（有多种表情）
- 点击有爆炸效果和惩罚文字
- 连击系统，连击越高越刺激
- 结束后根据得分获得评级称号

## 如何开始

1. 点击"开始制裁"按钮
2. 快速点击屏幕上出现的臭猪
3. 保持连击获得高分
4. 挑战"传说制裁者"称号！

## 评级系统

- 菜鸟制裁者: 0-19次
- 合格制裁者: 20-39次
- 熟练制裁者: 40-59次
- 精英制裁者: 60-79次
- 大师制裁者: 80-99次
- 传说制裁者: 100次以上
